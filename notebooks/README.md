# Notebooks del *pipeline*

Versiones HTML para revisión metodológica del proyecto <strong style="color:#0098cd">AgriVoice AI</strong>.

Esta carpeta contiene las versiones exportadas a HTML de los notebooks desarrollados durante el proyecto. Los notebooks ejecutables originales (`.ipynb`) no se incluyen en esta versión pública por motivos de privacidad, tamaño y control de exposición del código ejecutable.

Los archivos HTML permiten revisar la metodología, el flujo de trabajo, el código utilizado y los resultados principales de cada etapa del pipeline.



## Orden de revisión

1. `01_audio_preprocessing.html` 
   Preprocesamiento y estandarización de los audios.

2. `02_speech_to_text_asr.html` 
   Transcripción automática de los mensajes mediante ASR.

3. `03_nlp_classification_ner.html` 
   Clasificación de mensajes y extracción de entidades.

4. `04_nlp_normalization.html` 
   Normalización semántica y generación de la salida estructurada en formato JSON.



> [!NOTE]
>
> Estos archivos tienen finalidad documental y de revisión. No están pensados para su ejecución directa.
